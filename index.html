ParaView Lite

DocumentationGallery

A customizable Web client for ParaView

pvpython -m paraview.apps.lite

  • Releases

    ParaView Lite is client application that connect to a ParaView server to perform data processing and visualization. It is designed to be customizable in term of UI and visualization workflow. ParaView Lite is available inside ParaView and can be run via its Python environment (pvpython) using the command line listed above.
